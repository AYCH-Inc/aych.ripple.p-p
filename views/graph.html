Rippled Peer Graph

The Ripple Peer Graph is a visual representation of the rippled network.
Each node is a rippled and the links between the nodes are tcp connections.
A node's size corresponds to the number of connections it has and its color indicates the software version
The Ripple Peer Dashboard is built and maintained by Ripple Labs, based on data from the Ripple Network.
Information is provided "as is" and solely as an informative resource. Data may be delayed or incorrect.
